# 3차 프로젝트 1팀 회의록
- 날짜 2026년 03월 24일
- 팀명 : Prompt
- 참여자 : 정확,김재웅,박민성
## 오전회의 내용
- 자기소개하기
- 팀장 뽑기 ( 정확씨 당선 )
- 팀명 정하기 (닥트리오)
- 프로젝트 명 정하기 (prompt)
- 깃헙 저장소 복제 및 피그마 ERD 초대 수락
- 작업 요구 명세서 작성
## 작업 목표
- w3 벨리 데이터 오류 수정
## 작업 회고
- 2차 프로젝트에서 작성했던 READ.ME를 참고해서 3차 프로젝트용 READ.ME를 새로 작성하는 작업을 했다.  프로젝트 개요, 팀 구성, 역할 분담 같은 내용을 작성하고, 실제 코드 파일들을 보고 내용을 채웠다.
- ppt 작업 중 미흡한 부분들 채우고 잘 못 작성한 부분 수정중 팀원들 코드를 확인하여 살펴보고 다시 작성 하였다.
## 내일목표
- 발표 준비
## 팀 규칙
- ERD, 화면설계서, 명세서 는 기능이 변경 및 추가 될때마다 그때그때 바로 추가하고 팀원들 한테 고지할 것
- 다른사람이 작성하는 파일을 건드리거나 변경시 말하고 작업 시작하기 혹시 누가 작업중일 경우 잠깐 스탑 하고 다른일 먼저 처리
- 주석 달기 / 테스트 잘 실행하고 추가하기
- 야근이을 안하기 위해 최대한 노력하고 혹시나 야근이 필요할경우 야근 참석하기
- 싸우지말기
