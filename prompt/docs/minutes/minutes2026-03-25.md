# 3차 프로젝트 1팀 회의록
- 날짜 2026년 03월 24일
- 팀명 : Prompt
- 참여자 : 정확,김재웅,박민성
## 오전회의 내용
- 자기소개하기
- 팀장 뽑기 ( 정확씨 당선 )
- 팀명 정하기 (닥트리오)
- 프로젝트 명 정하기 (prompt)
- 깃헙 저장소 복제 및 피그마 ERD 초대 수락
- 작업 요구 명세서 작성
## 작업 목표
- w3 벨리 데이터 오류 수정
## 작업 회고
- 마이페이지를 레이아웃으로 해서 작업하다보니 csrf토큰 id가 겹치는 문제가 발생하여 한개의 토큰 아이디 삭제로 해결
- section태그를쓸때에는 h2~h6요소를 사용해야한다
- AI 도움말 버튼 /guide 페이지로 이동하도록 수정하였고 유튜브 요약 탭에 텍스트 직접 입력하는 기능을 추가 했슴.(이유는 EC2 서버에서 유튜브 자막요청 을 하면 ip가 aws 서버라는걸 알지만 유튜브 측에서 그걸 봇으로 인식해 막기때문)
## 내일목표
- 명세서들 확인하여 수정 및 보안 제출까지
## 팀 규칙
- ERD, 화면설계서, 명세서 는 기능이 변경 및 추가 될때마다 그때그때 바로 추가하고 팀원들 한테 고지할 것
- 다른사람이 작성하는 파일을 건드리거나 변경시 말하고 작업 시작하기 혹시 누가 작업중일 경우 잠깐 스탑 하고 다른일 먼저 처리
- 주석 달기 / 테스트 잘 실행하고 추가하기
- 야근이을 안하기 위해 최대한 노력하고 혹시나 야근이 필요할경우 야근 참석하기
- 싸우지말기
