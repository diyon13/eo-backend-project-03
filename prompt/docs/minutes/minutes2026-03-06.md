# 3차 프로젝트 1팀 회의록
- 날짜 2026년 03월 06일
- 팀명 : Prompt
- 참여자 : 정확,김재웅,박민성
## 오전회의 내용
- 자기소개하기
- 팀장 뽑기 ( 정확씨 당선 )
- 팀명 정하기 (닥트리오)
- 프로젝트 명 정하기 (prompt)
- 깃헙 저장소 복제 및 피그마 ERD 초대 수락
- 작업 요구 명세서 작성
## 작업 목표
- 피그마 화면 흐름 완성
- 코드 작성 시작
## 작업 회고
- 소셜 로그인 메일  구현할때 인증키 등 받아오는걸 까먹어서 다시설정하느라 애먹음
- repository 테스트 중 주에를 꼭달야된다
- 자동완성 기능 썻다가 믿을게 못된다는 것을 배웠다.
## 내일목표
- 주말이니까 최대한 푹쉬고오기
- 정확씨 월요일 예비군 화이팅 (m16들고 뺑뺑이 돌아요.)
## 팀 규칙
- ERD, 화면설계서, 명세서 는 기능이 변경 및 추가 될때마다 그때그때 바로 추가하고 팀원들 한테 고지할 것
- 다른사람이 작성하는 파일을 건드리거나 변경시 말하고 작업 시작하기 혹시 누가 작업중일 경우 잠깐 스탑 하고 다른일 먼저 처리
- 주석 달기 / 테스트 잘 실행하고 추가하기
- 야근이을 안하기 위해 최대한 노력하고 혹시나 야근이 필요할경우 야근 참석하기
- 싸우지말기
## 기술 스택
- spring boot
- JWT,JPA
- HTML,CSS,JS
- H2,SQL

- 민성 : 결제 하는거 어찌함? 진짜로
