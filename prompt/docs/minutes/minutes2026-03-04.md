# 3차 프로젝트 1팀 회의록
- 날짜 2026년 03월 04일
- 팀명 : Prompt
- 참여자 : 정확,김재웅,박민성
## 오전회의 내용
- 자기소개하기
- 팀장 뽑기 ( 정확씨 당선 )
- 팀명 정하기 (닥트리오)
- 프로젝트 명 정하기 (prompt)
- 깃헙 저장소 복제 및 피그마 ERD 초대 수락
- 작업 요구 명세서 작성
## 작업 목표
- 작업 요구 명세서에 많은 의견 제출하기
- 피그마 활용하여 기본 화면 구상하기
## 작업 회고
- 2차 프로젝트 덕분에 요구사항 명세서 및 화면설계서가 생각보다 매끄럽게 작성 됬으나 결제 라는 벽에 부딫혀 공부해야됨
- 미적감각이 부족하여 화면 설계서 작성할때 많은 고충을 격음
## 내일목표
- ERD 작성 시작하기
- 기능 명세서 작성하기
## 팀 규칙
- ERD, 화면설계서, 명세서 는 기능이 변경 및 추가 될때마다 그때그때 바로 추가하고 팀원들 한테 고지할 것
- 다른사람이 작성하는 파일을 건드리거나 변경시 말하고 작업 시작하기 혹시 누가 작업중일 경우 잠깐 스탑 하고 다른일 먼저 처리
- 주석 달기 / 테스트 잘 실행하고 추가하기
- 야근이을 안하기 위해 최대한 노력하고 혹시나 야근이 필요할경우 야근 참석하기
- 싸우지말기
## 기술 스택
- spring boot
- JWT,JPA
- HTML,CSS,JS
- H2,SQL
## 불만사항 접수
- 민성씨 화면에 정품인증(해결방안: 노력해볼께요 하지말 불가능에 가까움)
