# 3차 프로젝트 1팀 회의록
- 날짜 2026년 03월 12일
- 팀명 : Prompt
- 참여자 : 정확,김재웅,박민성
## 오전회의 내용
- 자기소개하기
- 팀장 뽑기 ( 정확씨 당선 )
- 팀명 정하기 (닥트리오)
- 프로젝트 명 정하기 (prompt)
- 깃헙 저장소 복제 및 피그마 ERD 초대 수락
- 작업 요구 명세서 작성
## 작업 목표
- 결제 연동 및 결제 페이지 뼈대 작업
- 관리자 페이지 뼈대 작업 시작
- ai채팅 페이지 천천히 작업 시작
## 작업 회고
- 결제 연동시 포트원 쪽에서 원하는 플랫폼 결제를 연동하여 js 쪽연 동할때 카카오나 토스 등 js 에 연동하여 사용해야 한다는 걸배웠고 아직 나머지 연동 및 오류가있어 수정작업이 필요함
- 인증 로직과 기능 로직의 역할을 분리하기 위해 컨트롤러를 나눴다
  관리자 로그인과 인증 처리는 AdminAuthController에서 담당하고,
  관리자 대시보드 및 사용자 관리와 같은 운영 기능은 AdminController에서 처리하도록 구조를 분리했다.
- Alan AI 대답이 {'type': 'continue', 'data': {'content': '글자'}} 형식인 걸확인해서 파싱 로직을  짰고,DB에  JSON이 저장되는 것도 parseContent()로 해결하였음
## 내일목표
- 주말이니 푹쉬고 모이기 맛있는거 많이 먹기
## 팀 규칙
- ERD, 화면설계서, 명세서 는 기능이 변경 및 추가 될때마다 그때그때 바로 추가하고 팀원들 한테 고지할 것
- 다른사람이 작성하는 파일을 건드리거나 변경시 말하고 작업 시작하기 혹시 누가 작업중일 경우 잠깐 스탑 하고 다른일 먼저 처리
- 주석 달기 / 테스트 잘 실행하고 추가하기
- 야근이을 안하기 위해 최대한 노력하고 혹시나 야근이 필요할경우 야근 참석하기
- 싸우지말기
